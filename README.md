##  About This Project

This is a test repository specifically created to learn advanced Git workflows, experiment with pull requests, and earn official GitHub achievements.

###  Badges I Want to Get:
*  **Pull Shark** - Merge 2+ Pull Requests.
*  **YOLO** - Merge a pull request without a code review.
*  **Quickdraw** - Close an issue or PR within 5 minutes of opening it.
*  **Starstruck** - Get 16+ stars on this repository.
